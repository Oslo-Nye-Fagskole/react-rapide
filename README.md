# react-rapide

Fast scaffolding for Vite-based React projects with TypeScript.

## Getting Started

React Rapide is designed for existing React projects created with Vite. Before installing React Rapide, create your React project, install its dependencies, and verify that the project runs successfully.

Example:

```bash
npm create vite@latest my-app -- --template react-ts
cd my-app
npm install
```

Once your project is ready, install React Rapide.

## How to use

Run

```bash
npm install @oslo-nye-fagskole/react-rapide --save-dev
```

Then show the menu from which you can choose different commands and get additional help:

```bash
npm run rr
```

(Requires Node 20.11.0 or newer.)
